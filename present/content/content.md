## The Game

- is played on a grid of variable size
- consists of up to 4 to 8 snakes<br/> on a single game board

...
- We haven't settle on game board size quite yet so that is subject to change.
  - Previous years we've used board sizes ranging from 15 to 50, so `truly anything is possibleâ„¢`
- Each tournament game will consist of 4 to 8 snakes on a single game board
